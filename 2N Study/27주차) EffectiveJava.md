## 20211107 김성민 프로
### NOTION https://www.notion.so/Effective-Java-3-E-6e8c1a05ab4c4ee08ccc6096ed919608
- 람다 클래스에서 this 는 외부클래스를 말한다
- 익명 클래스에서 this 는 자기 자신 클래스를 말한다
- 람다가 익명클래스보다 나은 점 중 가장 큰 특징은 간결함이다
- 메서드 참조란 Name space ( :: ) 을 사용하여 클래스 내의 메서드를 넘겨주는 것이다
  - 람다 참조 대신 해당 클래스의 메서드를 참조하여 넘겨주면 더 간결해진다
- 한정적 인스턴스 참조는 특정 인스턴스의 isAfter 메서드를 사용하여 해당 인스턴스로부터만 메서드를 참조하는 것을 말한다
- 특정 size 초과 시 eldest 를 제거하고 넣는 Map 등을 구현하는 경우, 표준 함수형 인터페이스 ( Function 내 BiPredicate 등 ) 를 사용한다
- 스트림 파이프라인은 특정 개수로 종단하는 종단연산으로 끝나야한다
  - 그 사이에 하나 이상의 중간연산이 있을 수 있다
